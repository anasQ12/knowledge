# معرفة — الموقع المدموج

ارفع كل محتويات هذا المجلد إلى مستودع GitHub (بنفس البنية) وسيعمل الموقع على GitHub Pages مباشرة.

- `index.html` — الصفحة الرئيسية (التصميم الجديد)
- `DiscoverYourself.dc.html` — اختبار اعرف نفسك (يعمل بالكامل)
- `Journey.dc.html` — صفحة رحلة
- `ConsultMe.dc.html` — صفحة استشرني (قيد الإعداد)
- `js/questions.js` — الأسئلة والنتائج (الملف الذي تعدّله)
- `vendor/` — مكتبات محلية يحتاجها التصميم الجديد (لا تحذفها)
- `_ds/` و `support.js` و `image-slot.js` — ملفات التصميم (لا تعدّلها)
